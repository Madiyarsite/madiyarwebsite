Портфолио

Я использую Bootstrap 4 последнюю версию на всех своих сайтах. Оно помагает быстро верстать и адаптировать сайт под мобильном экранам. Bootstrap пользуется многие популярные сайты.
Sass это компилятор для Css стилей. Sass ускоряет создание сайта в 4 раза. Этим компилятором ползуется многие популярные сайты. Если разработчику понадобится 100 cтрок кода, то с компилятором Sass всего 50. Это очень помагает созданию сайта.
Animation
CSS анимаций делает красивые анимаций блоков, картин. Меняет цвет и т.д не затрагивая Java script
Adaptation
Адаптация сайтов можно сказать обезательно. Так как многие заходят на сайт через смартфоны. Браузер подбирает сайтов и вставляет на самый вверх именно адаптивных сайтов.
